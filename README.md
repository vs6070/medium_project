# 🔐 Password Manager (GUI)

A simple and beginner-friendly **Password Manager** built using Python and Tkinter.
This application allows users to securely store, search, and generate passwords for different websites.

---

## 🚀 Features

* 🔑 Store website, username, and password
* 🔍 Search saved credentials
* 🎲 Generate strong random passwords
* 💾 Data stored locally in JSON file
* 🖥️ Simple and user-friendly GUI

---

## 🛠️ Technologies Used

* Python 3
* Tkinter (GUI)
* JSON (data storage)

---

```bash id="pm_readme_structure"

```

## ▶️ How to Run

1. Install Python (3.x)
2. Download or clone this repository
3. Open terminal / command prompt
4. Run the program:

```bash id="pm_run"
python password_manager.py
```

---

## 🎮 How to Use

1. Enter **Website name**
2. Enter **Username**
3. Enter or generate a **Password**
4. Click **Save** to store credentials
5. Use **Search** to retrieve saved data

## ⚠️ Disclaimer

* This project stores passwords in **plain text JSON format**
* It is intended for **learning purposes only**
* Do NOT use it for storing real or sensitive passwords
